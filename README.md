## Installing / Getting Started

```shell
yarn add prism-oceanic-next
```

```javascript
import 'prism-oceanic-next!css';
```

### Prerequisites

* Webpack

## Developing

### Setting up Dev

```shell
git clone https://github.com/blackxored/prism-oceanic-next
cd prism-oceanic-next
yarn
```
